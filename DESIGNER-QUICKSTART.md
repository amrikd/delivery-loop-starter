# Designer Quickstart

For before the workshop. Get set up so you can focus on designing, not debugging.

## Step 1 — Get the repo

Open Terminal (Spotlight → "Terminal") and paste:

```
git clone https://github.com/amrikd/delivery-loop-starter
cd delivery-loop-starter
./setup.sh
```

The script checks what's installed and tells you what's missing, color-coded.

## Step 2 — Install what's flagged

Most designers need to install these:

- **Figma desktop app** — [download](https://www.figma.com/downloads). Sign in with your Accenture Figma.
- **Claude Code** — [download](https://claude.com/claude-code). Sign in with your Accenture account.

Re-run `./setup.sh` after installing. You're ready when everything's green.

## Step 3 — Open Figma desktop and sign in

Keep it running. Claude talks to the desktop app, not the browser tab.

## That's it

At the workshop, you'll pair up, write specs with your dev partner, and start building. The rest you'll figure out in the room — that's the point.

If you want deeper context before then, skim `WRITING-SPECS.md`. But it's not required reading.
